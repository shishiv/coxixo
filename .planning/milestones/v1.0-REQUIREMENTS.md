# Requirements: Coxixo

**Defined:** 2026-01-17
**Core Value:** Frictionless voice input: hold a key, speak, release, paste.

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Core Experience

- [x] **CORE-01**: User can hold a global hotkey to record audio (push-to-talk) — Phase 1
- [x] **CORE-02**: User can release the hotkey to trigger transcription — Phase 3
- [x] **CORE-03**: App runs as system tray icon with minimal footprint — Phase 1
- [x] **CORE-04**: Tray icon visually indicates recording state (see UI-02 for design) — Phase 1
- [x] **CORE-05**: Transcription result is automatically copied to clipboard — Phase 3
- [x] **CORE-06**: User hears audio feedback when recording starts/stops (beep) — Phase 2
- [x] **CORE-07**: User receives error feedback if transcription fails (notification) — Phase 3

### Integration

- [x] **INTG-01**: Audio is captured from default system microphone — Phase 2
- [x] **INTG-02**: Audio is sent to Azure OpenAI Whisper API for transcription — Phase 3
- [x] **INTG-03**: API credentials (endpoint, key) are securely stored — Phase 1

### Configuration

- [x] **CONF-01**: User can customize the push-to-talk hotkey — Phase 4
- [x] **CONF-02**: User can configure Azure API credentials (endpoint, key) — Phase 3
- [x] **CONF-03**: Settings persist across app restarts — Phase 1

### Visual Identity (Brand Guide)

- [x] **UI-01**: Tray icon uses sound bar design (3 bars forming "C" shape) — Phase 4
- [x] **UI-02**: Tray icon is white/gray when idle, red with pulsing dot when recording — Phase 4
- [x] **UI-03**: Settings window follows dark theme (bg #1E1E1E, surface #252526) — Phase 4
- [x] **UI-04**: Settings window shows API connection status with latency indicator — Phase 4
- [x] **UI-05**: Color palette: Azure Blue #0078D4, Accent Green #00CC6A — Phase 4
- [x] **UI-06**: Typography: Segoe UI (system font) — Phase 4
